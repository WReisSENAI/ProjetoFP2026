# 🚀 Projeto Integrador: Sistema de Controle de Empréstimos RFID
Projeto Integrador Escola Filhinho Portilho 2º Ano - 2026

Este repositório contém a estrutura base para o desenvolvimento do sistema automatizado de monitoramento e controle de ativos (notebooks, tablets e livros) da nossa unidade técnica.

## 📋 Sobre o Projeto
O objetivo é solucionar o problema de extravio de equipamentos através da tecnologia **RFID (Radio-Frequency Identification)**, integrando hardware e software para um registro preciso de entradas e saídas.

## 🛠️ Tecnologias Utilizadas (Previsão)
* **Lógica & Backend:** Python ou Node.js (Desenvolvimento de Sistemas I)
* **Banco de Dados:** PostgreSQL/MySQL (Modelagem de Dados)
* **Hardware:** Sensores RFID e Microcontroladores (IoT)
* **Interface:** Django Web/Mobile (Programação de Aplicativos)
